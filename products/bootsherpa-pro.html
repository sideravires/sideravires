BootSherpa Pro

Open your apps at login

BootSherpa Pro runs a list of applications, files, folders, and URLs when you sign in to macOS. You choose what opens and set an optional delay for each item.

Climb faster. Start everything automatically.

Free on the Mac App Store. No account required.

Tired of this?

  • Opening the same apps by hand every morning
  • Clicking through Safari, Mail, Xcode, Terminal, or project folders one by one
  • No way to stagger launches — everything opens at once, or nothing does
  • Login Items in System Settings spread across apps, with no shared list

How it works

1

Add apps, files, folders, and URLs

Details

  • Applications, files, folders, and URLs in one list
  • Separate delay for each item (seconds)
  • Start at login and optional notifications
  • Uses current and older macOS login item APIs
  • Standard AppKit settings window — no account, no cloud

Typical use

If your day starts with the same set of tools — a browser, mail, Xcode, Terminal, Docker, or a project folder — you can define that set once and let macOS open it at sign-in.

BootSherpa Pro is a small utility for one job: managing what opens when you log in.
